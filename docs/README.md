# Docs index

`handoffs/` holds dated model handoff docs (e.g.
`HANDOFF_NWSL_MODEL_2026-05-28.md`) written at the end of each model-lab
session, summarizing what changed and what is still open. For active or
planned multi-agent runs, see `plans/` at the repo root (e.g.
`plans/2026-07-22-model-lab/`). For the two package READMEs, see
`CLAUDE.md` at the repo root, `nwsl-model/README.md` (Python model
package), and the root `README.md` (Next.js app).

`nwsl-model/docs/` (left in place, not moved) holds model-package-specific
notes: `season_validation_protocol.md` (the season holdout methodology) and
`research/soccer_modeling_public_research_2026-05-26.md` (public soccer
modeling literature review).
