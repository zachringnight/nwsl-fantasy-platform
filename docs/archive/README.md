# Documentation archive

This directory preserves superseded planning and research material that remains useful as historical context.

Archived documents do not control current implementation. Start with `plans/README.md` for active and completed execution packages.

## Contents

- `initial-product-planning/`: the original NWSL product prompt chain and its expanded outputs. These documents predate the implemented fantasy foundation and the current long-range roadmap.

Nothing in this archive should be treated as current production truth without checking the live repository and current plan.
