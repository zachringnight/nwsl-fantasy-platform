# Initial Product Planning Archive

This archive preserves the first NWSL fantasy product-planning bundle.

- `planning/` contains the concise prompt-chain source documents and usage guide.
- `outputs/` contains the expanded strategy, teardown, information architecture, flows, design system, game mechanics, technical architecture, screen specifications, build plan, and scaffold summary.

These files are not exact duplicates, so both sets are retained. They are superseded for implementation sequencing by `plans/2026-07-22-best-possible-nwsl-app/`.

Use them only for historical product and design context. Reconcile any reused idea with the current roadmap, schema, routes, data providers, and production behavior.
